# 🚀 Dev Bootstrap

> My personal EndeavourOS GNOME development environment.

A reproducible Linux workstation setup for AI, Robotics, Competitive Programming, and Software Development.

---

## ✨ Features

- 🖥️ EndeavourOS + GNOME
- 🐚 Zsh + Starship
- 👻 Ghostty + Terminator
- 📂 Yazi
- 🌳 Lazygit
- 🐳 Docker
- 🐍 UV Python
- 📦 Mise
- ⚡ Fastfetch
- 🎨 Catppuccin
- 💻 VS Code
- 🔧 Modern CLI utilities
